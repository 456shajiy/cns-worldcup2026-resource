# cns-worldcup2026-resource

本仓库用于归档和发布多个独立 HTML 页面资源，不针对任何特定域名或具体网站。所有页面均以纯前端方式构建，旨在提供结构化的信息展示与交互功能。

## 项目简介

该项目主要收集、整理并公开一系列自包含的 HTML 文件，内容涵盖信息展示、工具页面及静态资源集合。仓库内页面均为独立运行，无需后端服务支持，可直接在浏览器中打开使用。

## 目录结构

```
cns-worldcup2026-resource/
├── pages/               # 存放各 HTML 页面的主目录
│   ├── example-page-1/  # 示例页面文件夹（实际命名以内容为准）
│   ├── example-page-2/
│   └── ...
├── assets/              # 公共静态资源（CSS、JS、图片等）
├── docs/                # 辅助文档或说明文件（可选）
└── README.md            # 本文件
```

各页面文件夹内通常包含一个 `index.html` 及所需资源文件，可独立部署或离线查看。

## 页面归档说明

- 所有 HTML 页面均为独立归档，彼此无依赖关系。
- 页面中不包含任何指向外部特定域名或商业网站的链接。
- 归档内容以通用信息展示或工具功能为主，不针对任何特定实体或活动。
- 页面使用标准 HTML5 语法，兼容主流现代浏览器。
